# Gestor de Tarefas

Aplicação web de gerenciamento de tarefas desenvolvida para organizar atividades do dia a dia de forma simples, prática e intuitiva.

## 🌐 Projeto Online
Acesse aqui:  
👉 https://gestor-de-tarefas-nicolas2.web.app

## 📌 Funcionalidades
- Adicionar novas tarefas
- Listar tarefas cadastradas
- Marcar tarefas como concluídas
- Remover tarefas
- Interface simples e fácil de usar

## 🛠️ Tecnologias utilizadas
- Angular
- TypeScript
- HTML
- CSS
- Firebase Hosting
